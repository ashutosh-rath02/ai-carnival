# ai-carnival

A collection of practical, hands-on AI experiments — each one a small,
self-contained project you can run end to end.

## Projects

- **[ai-ocr-field-test](ai-ocr-field-test/)** — a field test of 4 OCR /
  document-AI tools (PaddleOCR, Gemini 3.5 Flash, Mistral OCR, and Baidu
  Unlimited-OCR) on messy real-world documents. Runs extraction, captures latency,
  scores quality, and generates a comparison report. Headline finding: on messy
  inputs the tools fail in completely different ways — one blanks, one
  hallucinates, one refuses, one just works.

_More experiments to come._
